Руководство пользователя

Для работы программного комплекса необходима операционная система Windows. Активизация пособия производится с помощью запуска на исполнение файла «KTO.exe». При запуске  данная программа разворачивает на весь экран свое главное окно.

Главное окно данной программы содержит 2 окна: левое и правое. Правое окно присутствует всегда, а левое - появляется при поднесении мыши к левому краю главного окна и исчезает при отведении.

В левом окне в виде дерева представлено содержание электронного пособия в виде иерархического дерева. Вложенные разделы подсоединяются к родительскому разделу и не видны. Для доступа к вложенным пунктам необходимо щелкнуть левой клавишей мыши на знаке «+» слева от родительского раздела. После этого появятся вложенные пункты и знак «+» заменится на знак «-», при нажатии на него мышью подветвь разделов будет свернута. Может существовать иерархия до шести уровней вглубь.

При нажатии левой клавиши мыши на названии подраздела в левом окне, в правом будет показан сам материал. Правое окно отображает гипертекст или flash-презентации точно так же, как и Internet Explorer в Windows. Данные гипертекста загружаются из файлов с расширением htm, располагаемых в одном каталоге с данной программой. Если гипертекст или flash-презентация будет отсутствовать в каталоге программы, то в правом окне появится сообщение «невозможно загрузить страницу».

Изменить размеры окон можно следующим образом:

- главное окно с помощью иконок масштабирования окна и за края окна,

- дочерние окна с помощью перемещения мышью вертикального разделителя.

При изменении размеров главного окна автоматически изменяются размеры дочерних окон.

В программе используются следующие кнопки, располагающиеся в верхней части главного окна.

 - перейти к предыдущему окну (в списке последних используемых страниц);

 - перейти к следующему окну (в списке последних используемых страниц);

 - прервать загрузку файла;

 - обновить текущую страницу;

 - сделать левое окно плавающим (при поднесении мыши автоматически появляется окно содержания); после нажатия на эту кнопку вид ее заменяется на ;

 - зафиксировать положение левого окна; после нажатия на эту кнопку вид ее заменяется на .

Содержание состоит из следующих разделов.

- титульный лист;

- об электронном пособии – краткие сведения об электронном пособии;

- руководство пользователя (данная информация);

- курс обучения – по шагам демонстрируются возможности данного пособия и работа с ним;

- разделы лабораторных работ;

- общий список литературы для самостоятельной подготовки;

- список терминов.

Содержание лабораторных работ состоит из следующих пунктов:

- цель работы;

- теоретическая часть – может содержать в себе другие подпункты;

- используемые алгоритмы или алгоритм работы;

- рассмотренные примеры работы алгоритмов;

- примеры для самостоятельного решения;

- задание;

- примечания;

- литература для самостоятельной подготовки;

- вопросы для самоконтроля;

- тестовые задания – набор тестов для контроля знаний обучаемого;

- презентация (представление алгоритма с помощью технологии flash);

- программная модель алгоритмов – исполняемая программа.

В каждой из лабораторных работ существует пункт «Тестирование». При нажатии на этой строке левой клавишей мыши появляется диалог, запрашивающий ответ на поставленный вопрос с несколькими вариантами ответов. Необходимо выбрать один или несколько ответов, в зависимости от решения (выбор делается слева от варианта ответа – отметка галочкой). Перейти к следующему вопросу можно перейти с помощью кнопки «Далее>». По мере выполнения тестовых заданий текущий результат будет отображаться:

- в течение нескольких секунд в правом нижнем углу окна диалога в виде зеленого (правильный ответ) или красного (неправильный ответ) прямоугольников;

- вверху диалогового окна в виде строки «Пользователь ответил на i вопросов из N».

Если по результатам тестирования оказалось, что обучаемый не проявил должный уровень знаний и умений, то обучаемому необходимо обратиться к теоретическому материалу данного электронного пособия, рассмотреть книги из списка дополнительной литературы или рассмотреть по шагам работу поставленной задачи  с помощью программной модели, прилагаемой к лабораторной работе.

В окне последнего теста появится кнопка «Готово», предназначенная для завершения тестирования. Для прекращения теста существует кнопка «Отмена». Итоговый результат тестирования будет отображен в виде сообщения. Далее этот результат будет отображаться в заголовке главного окна.

С помощью электронного пособия возможен запуск программных моделей, показывающих работу описанных в лабораторных работах алгоритмов. Для вызова этих моделей необходимо щелкнуть левой клавишей мыши по строке «Программная модель алгоритма» в текущей лабораторной работе в окне содержания (левое окно).

В самом материале могут встречаться ссылки на разделы или подразделы, к которым можно перейти по нажатию на ссылке левой клавишей мышью.
