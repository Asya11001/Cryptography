Простыми числами называются натуральные числа, которые делятся только на себя или на единицу.

Решето Эратосфена — это древнейший способ выписывания простых чисел. Производится итеративное вычеркивание кратных чисел для простых найденных чисел.

 

Наибольший общий делитель чисел а и b - это наибольшее целое число d, на которое и а, и b делятся: d = НОД(а, b).

Числа а и b взаимно простые, если их НОД(а, b) = 1.

 

Алгоритм Эвклида предназначен для вычисления наибольшего общего делителя двух натуральных чисел.

 

Расширенный алгоритм Евклида позволяет находить для заданного модуля обратный элемент для числа a.

Если числа a и n взаимнопросты, то для числа a существует единственный обратный элемент a-1, при этом числа a и a-1 удовлетворяют условию (a * a-1) mod n =1.

Задача для чисел Мерсенна – определение среди этих чисел ряда простых чисел.
