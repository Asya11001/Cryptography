Лабораторная работа

Схема шифрования Эль-Гамаля

 

Цель работы: освоение практических методов использования и собственной программной реализации алгоритмов ассиметричного шифрования на примере схемы Эль-Гамаля.

 

Описание работы.

Схема Эль-Гамаля, предложенная в 1985 г., используется для шифрования и для цифровых подписей. Безопасность схемы Эль-Гамаля обусловлена сложностью вычисления дискретных логарифмов в конечном поле. В реальных схемах шифрования используется в качестве модуля Р простое число размерностью в [512; 1024] бит.

Выберем большое простое число P и два больших целых числа G и X, меньших P: G < Р, X < Р. Выберем случайное целое число К, удовлетворяющее условиям:

1 < К < Р-1,

НОД(K, P-1)=1 (взаимно простые).

Числа P и G являются открытой информацией, а X и K – секретные.

Далее вычислим открытый ключ Y = Gx mod P.

Для передачи сообщения М вычисляем два числа, образующих шифротекст:

a = GK mod P

b = YK M mod P.

В результате шифротекст по длине в 2 раза больше открытого текста М.

Возведем уравнение для a в степень x: ax  = Gxk mod P.

Изменим выражение b: так как Y = Gx mod P, то b = YK M mod P = (Gx)k M mod P = Gxk M mod P. Так как ax  = Gxk mod P, то b = Gxk M mod P = ax M mod P. Тогда восстановим открытый текст M: M = b / ax  mod P.

Для нахождения M нужно решить сложное сравнение b = ax M mod P или ax M/b  = 1 mod P.

Для решения сложного сравнения вида a*x ≡ b mod n применим метод замены: z = x/b. Тогда a*z ≡ 1 mod n, где z является обратным элементом для a. Число z определяется с помощью расширенного алгоритма Евклида.

Тогда из z = x/b  => x = z*b mod n.

 

Применим данную схему для нахождения открытого текста M в axM = b mod P. Обозначим за z выражение z = M/b, которое будет обратным элементом для ax по модулю P: ax z ≡ 1 mod P. Тогда M = z*b mod P = a-x*b mod P.

Механизм использования схемы Эль-Гамаля.

Предлагается реализовать данные алгоритм для чисел размерностью в машинное слово (до 32 бит – 232 комбинаций), задавая в качестве исходного открытого текста буквы русского алфавита, и выбирая модуль, соизмеримый с размером алфавита (если модуль будет меньше, то результат расшифровки будет отличаться от исходного открытого текста).

Пусть пользователь А собирается передать пользователю В сообщение в зашифрованном виде, используя схему Эль-Гамаля. Пользователь А выступает в роли отправителя сообщения, а пользователь В - в роли получателя. Элементы схемы Эль-Гамаля должен формировать получатель сообщения - пользователь В. Рассмотрим последовательность действий пользователя В и пользователя А.

Действия пользователя B.

1.   Выбирает произвольное большое простое числа Р.

2.  Выбирает случайное G, G < P и секретный ключ X, X < P.

3. Вычисляется открытый ключ Y = Gx mod P.

4. Стороне A передаются числа G, P и Y.

Действия пользователя A, передающего пользователю В сообщение М.

5.  Пользователь А разбивает исходный открытый текст М на блоки, каждый из которых   может   быть   представлен в виде последовательности чисел Mi = 0, 1, …, n-1.

6. Случайно выбирается K, удовлетворяющее условиям

1 < К < Р-1 и НОД(K, P-1)=1.

7. Вычисляется первая часть шифротекста: a = GK mod P.

8. Пользователь А шифрует текст, представленный в виде последовательности чисел bi по формуле bi = YK Mi mod P (схема Горнера) и отправляет криптограмму (a, b1, b2, …, bi,…) пользователю В.

Действия пользователя B.

8. Находит обратный элемент для ax по модулю P.

9. Расшифровывает Mi = bi*a-x mod P, .

В результате будет получена последовательность чисел Мi, которые представляют собой исходное сообщение М.

Действия пользователя В.

1. Выберем P = 37, G = 2, секретный ключ X = 8.

2. Вычисляем открытый ключ Y = Gx mod P = 28 mod 37 = 34.

3. Передача А числа P, G, Y.

Действия пользователя A.

4. Выбирает число K = 7. Оно 1 < К < Р-1 и НОД(K, P-1)=1.

5. Вычисляется a = GK mod P = 27 mod 37 = 17.

6. Шифруем M и передаем стороне B:

M0= 30: b0 = M0YKmod P = 30*347 mod 37 = 30*33 mod 37= 28.

M1= 2: b1 = M1YKmod P = 2*347 mod 37 = 2*33 mod 37= 29.

M2= 21: b2 = M2YKmod P = 21*347 mod 37 = 21*33 mod 37= 27.

Действия пользователя В.

7. Находит обратный элемент для ax по модулю P: ax *a-x ≡1 mod P.

ax = ax mod P = 178 mod 37 = 33. Его обратным элементом является a-x = 9 (расширенный алгоритм Евклида).

8. Расшифровываем шифротекст:

b0= 30: M0 = b0a-x mod P = 28*9 mod 37 = 30*33 mod 37= 30.

b1= 2: M 1 = b1 a-x mod P = 29*9 mod 37 = 2*33 mod 37= 2.

b2= 21: M 2 = b2 a-x mod P = 27*9 mod 37 = 21*33 mod 37= 21.

Восстановили сообщение “ЭБФ”.
