Захаров В.М., Эминов Б.Ф. Лабораторный практикум по алгоритмам шифрования: Учебное электронное пособие.

 

Казанский Государственный Технический Университет

им. А.Н. Туполева

 

Кафедра КС

2006 год

 

Учебное электронное пособие в виде программного комплекса обучающих средств предназначено для проведения лабораторного практикума по дисциплинам:

- «Математические основы криптологии»,

- «Криптографическая защита информации»,

- «Криптографические методы и средства обеспечения информационной безопасности»

для студентов:

- дневного,

- вечернего и

- заочного обучения

специальностей:

- 090103 (075300) – «Организация и технология защиты информации»,

- 090104 (075400) – «Комплексная защита объектов информатизации» и

- 090106 (075600) - ''Информационная безопасность телекоммуникационных систем''.

Комплекс содержит теоретический материал по лабораторным работам, примеры решения задач, вопросы для контроля и самоконтроля знаний, презентации на технологии flash, а также примеры программных моделей, демонстрирующие работу алгоритмов из лабораторных работ в виде графиков, схем и трассировочных таблиц, позволяющие обучаемым по шагам наблюдать вычисление поставленной задачи и проверять правильность работы собственных вариантов решения.

Представлены лабораторные работы по следующим темам:

1. алгоритм ассиметричного шифрования RSA;

2. алгоритм ассиметричного шифрования Эль-Гамаля;

3. цифровая подпись по схеме RSA;

4. цифровая подпись по схеме Эль-Гамаля;

5. цифровая подпись стандарта ГОСТР 34.10-94;

6. цифровая слепая подпись; схема Чаума;

7. открытое распределение ключей на базе схемы Диффи-Хеллмана.

Дополнительно, для самостоятельной подготовки, предоставляется flash-презентация и программная модель работы алгоритма шифрования ГОСТ 28147-89.

Вышеперечисленные лабораторные работы базируются на следующих лабораторных работах для курса «Математические основы криптологии», приведенных в данном учебном пособии:

1. нахождение наибольшего общего делителя с помощью алгоритма Евклида;

2. нахождение обратных элементов по расширенному алгоритму Евклида;

3. реализация схемы Горнера для степенных целочисленных функций;

4. поиск простых чисел с помощью модифицированного «решета Эратосфена»;

5. разложение чисел методом проб на два множителя (метод Бухштаба);

6. проверка чисел Мерсенна на простоту.

 

Количество экземпляров: 1 шт.

Размер комплекса в установочном пакете: 3 Мбайта.

Установленный комплекс занимает 8 Мбайт.

 

Используемый носитель информации: CD-Disk 700 Мбайт или любой другой носитель, способный хранить указанный выше объем информации под установочный пакет учебного электронного пособия.

Необходимые средства работы: персональный компьютер под управлением ОС Windows, при наличии средств просмотра гипертекстовой информации. Дополнительно в ОС должен быть установлен проигрыватель или com-объект для проигрывания flash-файлов (flash-презентаций).
