Разложение числа методом проб на два сомножителя

(Метод Бухштаба)

 

Цель работы: освоение метода разложения чисел на два сомножителя.

 

Описание работы.

Поясним используемые обозначения. N – множество натуральных чисел. Все операции производятся над целыми числами. Под формулой  будет пониматься весь ряд целых чисел, лежащих в диапазоне от r до k включительно, и который может принимать переменная i. ]k[ - взятие от дробного числа k только целой части, без округления.

Пусть задано натуральное число n, n≥9, которое необходимо разложить на два сомножителя.

Тогда итеративно вычисляем элементы Ns:

N0=n

N1=N0+1

N2=N1+3

N3=N2+5

…..

Ns=Ns-1+2s-1

На каждом шаге к Ns прибавляются числа в порядке возрастания: для вычисления Ns к Ns-1 прибавляется число (2s-1).

Прибавляемые к Ns числа являются членами арифметической прогрессии:

- начальный элемент равен a0=1,

- последний - as=2s-1,

- число членов – m=s

- шаг прогрессии равен 2.

Тогда сумма чисел прибавленных к n равна (по формуле суммы арифметической прогрессии) Sпрогрес= (a1 +as )m/2=(1+2s-1)s/2=2s2/2= s2.

Тогда элемент Ns равен Ns = N0 + Sпрогрес = n + Sпрогрес= n + s2. Пусть Ns = t2, где t – некоторое натуральное число, тогда Ns = t2= n + s2  Þ  n = Ns - s2 = t2- s2.

Если при некотором s будет возможно извлечение квадратного корня нацело из Ns= t2, то исходное число n разлагается как n=(t2-s2)=(t-s)(t+s), где числа (t±s) могут являться простыми.

Если в диапазоне для s: 1≤s≤](n-9)/6[ не будет найдено t2, где из t2 нацело извлекается квадратный корень, то процесс поиска двух сомножителей числа n завершается.
